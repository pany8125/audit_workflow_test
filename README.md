# 审计报表代码库

## 文件夹层级规则

- 一级目录名固定为【审计报表】
- 二级目录名为脚本对应时间,格式为YYYYQQ，例如【2018Q3】
- 三级目录名为各个业务线名称，例如【消费金融】
- 四级目录名为具体需求，格式为需求id+需求描述，例如【101_小白条日汇报表需求】
